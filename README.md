## Overview

### This is Mandar Apte. ExperienceServices is my initiative to assist developers in accelerating their coding workflows by utilising reusable code that saves time and effort for each iteration. You can use this Swift package without any hassle. A small initiative to share my most used swift helpers in public release.

## Features

- Swift Property Extensions for Array, Double, Int, NSLayoutConstraint, String, UIColor, UIDevice, UIImage, UIScreen, UIStackView, UIViewController
- Xcode and Swift Helpers and Utilities
- Time Saver Code Samples to Use, Reuse and Share Between Xcode Projects

### Read Complete Documentation Here: https://mandarapte.com/apple/new-public-github-repository-release-experienceservices-framework-swift-package/

## Requirements

### - iOS 17.0+ / iPadOS 17.0+ / Mac Catalyst 17.0+
### - Swift 6.2+

## Installation

### Swift Package Manager

Add ExperienceServices to your `Package.swift` file:

```
https://github.com/mandarapte/ExperienceServices.git
```

Or add it directly in Xcode:

1. File > Add Package Dependencies
2. Enter the repository URL
3. Select the version you want to use
